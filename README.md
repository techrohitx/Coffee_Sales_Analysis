# ☕ Coffee Sales Analysis using Excel

## 📌 Project Overview
This project focuses on analyzing coffee sales data using Microsoft Excel to extract meaningful business insights. It includes data cleaning, transformation, and visualization to understand sales performance and trends.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-selling coffee products
- Track revenue trends over time
- Compare sales across different categories/locations
- Build an interactive dashboard for decision-making

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Excel Formulas

---

## 📊 Key Features
- Data Cleaning and Preprocessing  
-  Sales Trend Analysis (Daily/Monthly/Yearly)  
- Top-Selling Products Identification  
-  Revenue Calculation  
-  Category/Region-wise Analysis  
-  Interactive Dashboard  

---

## 📂 Dataset
The dataset contains the following fields:
- Order Date  
- Product Name  
- Category  
- Quantity Sold  
- Unit Price  
- Total Sales  
- Location
